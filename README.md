# Todo-list
 
Esse é um pequeno projeto, feito para que eu pudesse praticar algumas funcionalidades do **Javascript**, como a manipulação do **DOM** e o uso do `localStorage()`.

Aqui aprendi a manipular o `localStorage()` em conjunto com o **JSON** para setar e buscar informações guardadas no armazenamento do site. Assim, como também pude aprender a selecionar e manipular apenas os input's do tipo "checkbox" que estão marcados, tal como varrer um array com tags **HTML** e adicionar um conteúdo à elas. Apesar de ser um simples projeto, me proporcionou bastante aprendizado e melhoria da minha lógica de programação.
